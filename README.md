# PawShield

AI-powered pet insurance guidance for smarter protection.

A simple Next.js MVP for Canadian dog and cat owners to compare pet insurance options with an educational PawShield Protection Score.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## MVP flow

Home page -> pet profile form -> results page with breed risks, score, recommended coverage, and provider quote buttons.

Provider quote links use official provider quote pages and can be replaced with approved partner links later.
